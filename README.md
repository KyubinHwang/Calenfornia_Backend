### 핵심 기능
- 홍익대학교 클래스넷 계정을 이용한 학생인증 및 로그인 기능
- 내가 듣는 수업 필터링(과목, 교수, 분반) 기능
- 전체 수업 일정에 대한 캘린더 기능
- 수업에 대한 일정 정보 추가 기능 (과제, 시험 등)
- 모든 User 일정 정보 열람 및 수정 기능

### 부가 기능
- 각각 User에 대한 todoList
- 일정에 대해 소통할 수 있는 일정마다의 채팅방 기능
